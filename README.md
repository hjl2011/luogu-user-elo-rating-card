1. 将 `config-example.json` 重命名为 `config.json`。
2. 将 `config.uid` 改成您的洛谷 uid。
3. 将 `config.lang` 改成您使用的语言。（`zh-CN`、`zh-TW`、`en-US`）
4. 运行 `npm install`。
5. 运行 `npm start`。
6. 得到 `luogu-elo-rating.svg`。
